# 00 — Preface

This project is an open research experiment.
It is not an LLM, does not aim to imitate an LLM, and does not start from any
established architecture.

The core idea is simple to state and hard to prove:

> knowledge might be representable as a compact harmonic structure,
> and language as discrete sampling of a continuous dynamics.

These documents collect reasoning as it emerges —
doubts, course corrections, architectural decisions, open hypotheses.
They are not technical documentation. They are a thinking diary.

Each file is numbered by order of appearance, not by importance.
